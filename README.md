# Atlas
My personal AI

A JARVIS-style personal AI: voice in/out, a live chat UI (`index.html`), and
a backend (`server/`) that connects it to Gmail, Google Calendar,
RevenueCat, Buffer, Instagram, Meta Ads, device location, and web lookups.

- **Run it:** see `server/README.md`.
- **Wire up a connector:** see `CONNECTORS.md` for exactly which credential
  to get and where.
- **Roadmap:** see `PLAN.md`.
