# Atlas
My personal AI

A personal AI: voice in/out, a live chat UI (`index.html`), and a backend
(`server/`) that connects it to Gmail, Google Calendar, device location, and
web lookups — all visible in one place via the **🔌 WIRES** panel.

- **Run it:** see `server/README.md`.
- **Wire up a connector:** see `CONNECTORS.md` for exactly which credential
  to get and where.
- **Roadmap:** see `PLAN.md`.
